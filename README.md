# Saathi.

**Always by your side.**

A multilingual digital banking companion that makes financial services simple, safe, and accessible for first-time users.

## Stack

- React + Vite
- Tailwind CSS v4
- React Router
- Framer Motion
- react-i18next (EN / HI / PA / TA / BN / MR)
- Zustand
- React Leaflet
- Web Speech API
- React Hot Toast
- React Hook Form

## Run

```bash
npm install
npm run dev
```

## Pages

Landing → Onboarding (language wow moment) → Dashboard → Balance, Transfer, Learn, Safety, Nearby, Voice, Help, Settings

## Accessibility

Keyboard navigation, screen-reader labels, dark mode, high contrast, font scaling, voice reading, semantic HTML, visible focus states.
